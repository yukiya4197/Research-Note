📷 AIカメラ MAIN

画像認識などのAI技術を用いて、カメラデバイスに知覚や判断能力を持たせる研究です。

タイトル:

AIカメラを用いた商店街の人流計測と活性化

想定シナリオ:

商店街に設置したカメラから、通行人の数や属性（年代・性別など）を自動で計測・データ化し、イベントの効果測定やマーケティングに活用します。

技術詳細:

  • モデル: YOLOなどの軽量な物体検出アルゴリズム
  • 言語・ライブラリ: Python, OpenCV, PyTorch/TensorFlowなど
  • ハードウェア: Raspberry Pi等のエッジデバイス、またはPC連携ウェブカメラ

💻 Web開発 MAIN

HTML/CSSやJavaScriptを用いて、自分だけのウェブサイトやウェブアプリを構築する方法を研究します。

タイトル:

はじめてのWebサイト作成ガイド

学習概要:

Webサイト構築の2つのアプローチ（Wix/WP等のGUIツール活用と、HTML/CSSのコーディング）を比較し、コーディングによる自由度の高い制作方法を学びます。

技術詳細:

  • 基礎技術: HTMLによる構造化、CSSによるデザイン
  • ツール・環境: ブラウザの開発者ツール、Visual Studio Code
  • 学習手法: 生成AIをメンターとした段階的な対話型学習
  • 公開: GitHub Pagesなどを利用した公開・デプロイ

🚀 発展系のアイデア

✨ お手製プリクラ ADVANCED

AIカメラ技術とWeb技術を応用して、顔認識フィルターやメイク機能をブラウザ上で動かすアプリを作成します。

タイトル:

WebARで作るビューティーフィルターアプリ

仕組み・概要:

AIによる骨格認識を用いて、美肌、デカ目、小顔補正や、チークの色・濃さを調整できる「プリクラ風機能」をブラウザだけで実現します。さらに、シャッターボタンを実装すれば、実際に加工した画像をスマホに保存できるアプリへと発展させることも可能です。

技術詳細:

  • AI顔認識: MediaPipe (Face Landmarker) によるリアルタイムな顔パーツ検出
  • 画像処理: WebGL と カスタムシェーダー(GLSL) を用いた高速な描画や顔の変形
  • UI制御: HTMLのレンジスライダーとカラーピッカーを使った直感的な操作連携
